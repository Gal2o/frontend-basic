- ### 선택자 우선순위
  - #### 우선순위란, `같은 요소`가 여러 선언의 대상이 된 경우, `어떤 선언의 css 속성`을 `우선 적용`할지 결졍하는 방법
    - #### `점수가 높은 선언`이 우선순위!
    - #### 점수가 같으면, `가장 마지막에 해석된 선언`이 우선순위!
    - #### `인라인 style 방식은 지양`하자

  - #### 2. `.list` (클래스: 10) + `li` (태그: 1) + `:hover` (가상 선택자 = 클래스: 10) = 21
  - #### 3. `.box` (클래스: 10) + `::before` (가상 요소 = 태그: 1) = 11
  - #### 4. `#submit` (id: 100) + `span` (태그: 1) = 101
  - #### 5. `header` (태그: 1) + `.menu` (클래스: 10) + `li` (태그: 1) + `:nth-child` (가상 선택자 = 클래스: 10) = 22
  - #### 6. `h1` (태그 :1) = 1
  - #### 7. `:not` (부정 선택자 != 클래스: 0) + `.box` (클래스: 10) = 10
